# jfaundez.tech - Soluciones Tecnológicas 360°

## 💻 Sobre el Proyecto
Página web personal y portafolio interactivo desarrollado para presentar servicios de Desarrollo Web, Inteligencia Artificial, Marketing de Datos y Soporte Técnico Empresarial. Diseñado bajo una estética moderna, responsiva y orientada a la conversión.

## ✨ Características Principales
*   **Arquitectura:** HTML5, CSS3 Nativo y Vanilla JavaScript.
*   **Diseño UI/UX:** Patrón Glassmorphism, paleta Cyber-Tech (Cyan y Púrpura), y diseño vectorial.
*   **SEO Técnico:** Intervenciones de metadatos integradas, OpenGraph cards para previsualización en redes sociales, Sitemap dinámico y robots.txt.
*   **Analítica:** Google Analytics 4 (GA4) y Google Tag Manager (GTM) incorporados nativamente en el renderizado inicial y secundario.
*   **Optimización Mobile First:** Grillas adaptativas asimétricas (Bento Layout) rediseñadas específicamente para pantallas en dispositivos móviles y tablets.

## 🚀 Despliegue Local
Debido a su estructura puramente del lado del cliente (Front-End Estático), este proyecto no necesita compiladores ni entornos de Servidor Node.js complejos para correr localmente.

1. Clona el repositorio.
2. Abre \`index.html\` en cualquier navegador moderno.
3. Para una experiencia más certera de APIs y tipografías, puedes usar un servidor local por puerto como \`Live Server\` de VS Code.

## 📦 Estructura de Proyecto
*   \`/assets\` - Imágenes pesadas, vectores SVG, favicons y fotografías de portafolio y previsualización (OG Image).
*   \`/css\` - Lógica de estilos en cascada modulares. Variables raíz y Keyframes globales.
*   \`/js\` - JavaScript puro. Efectos de tipeo (Typewriter), manejo visual, animaciones modales y sticky headers.
*   \`sitemap.xml\` & \`robots.txt\` - Guías estáticas para Googlebot.
